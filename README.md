# 🚀 Job Portal (MERN Stack)

A full-stack **Job Portal Web Application** built using the **MERN Stack (MongoDB, Express, React, Node.js)**.
This platform allows recruiters to post jobs and users to browse and apply for jobs easily.

# ✨ Features

### 👤 User Features

* User Registration & Login
* Browse all jobs
* Search and filter jobs
* View job details
* Apply for jobs
* View applied job status

### 🏢 Recruiter / Admin Features

* Create and manage companies
* Post new jobs
* Edit job details
* View job applicants
* Manage posted jobs

# 🛠 Tech Stack

### Frontend

* React.js
* Redux Toolkit
* Tailwind CSS
* DaisyUI
* Axios
* Framer Motion

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

# ⚙️ Installation

### 1️⃣ Clone Repository

```
git clone https://github.com/AshrafAbid078/job-portal.git
```

### 2️⃣ Install Dependencies

Backend

```
cd backend
npm install
```

Frontend

```
cd frontend
npm install
```

---

# 🔐 Environment Variables

Create `.env` file inside **backend**

```
PORT=5000
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_secret
CLOUD_NAME=your_cloudinary_name
API_KEY=your_cloudinary_key
API_SECRET=your_cloudinary_secret
```

---

# ▶️ Run Project

Backend

```
npm run dev
```

Frontend

```
npm run dev
```
